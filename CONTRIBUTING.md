# Contributing

## Branches
- main: stable releases only
- dev: integration branch
- feature branches: feat/*, fix/*, perf/*, docs/*, chore/*

## Commit messages
Use Conventional Commits:
feat:, fix:, perf:, docs:, refactor:, chore:

## Definition of Done
- No console errors
- No layout reads in the rAF loop
- Docs updated if behaviour changes
