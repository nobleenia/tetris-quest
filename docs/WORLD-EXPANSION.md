| Block | Visual | Effect |
|-------|--------|--------|
| **Mirror Block** | 🪞 | Reflects the current piece (horizontal flip) when landed adjacent |
| **Magnet Block** | 🧲 | Pulls pieces toward it within 2 cells |
| **Rainbow Block** | 🌈 | Wild — counts as any color for color-match objectives |
| **Ghost Block** | 👻 | Fades after 10 seconds, creating a delayed gap |
| **Chain Block** | ⛓️ | Connected — clearing one clears all connected chain blocks |
| **Clock Block** | ⏰ | Adds 5 seconds to timer when cleared |
| **Coin Block** | 🪙 | Drops bonus coins when cleared |
| **Skull Block** | 💀 | Penalty — must be cleared within 5 lines or triggers damage |
| **Crystal Block** | 💎 | Requires 2 clears to break (first crack, then clear) |

## Advanced Worlds (Worlds 61–80) — Expert Players

### Mirror Dimension (Worlds 61–65)
- Mirrored/inverted controls, upside-down board, rotated views
- Introduces F-piece and asymmetric challenges

### Speed Run Zone (Worlds 66–70)
- Ultra-fast gravity, tiny time limits, perfect-play requirements
- Leaderboard-focused for competitive play

### Puzzle Rooms (Worlds 71–75)
- Pre-placed boards with specific solutions
- Clear in X moves. Chess-like strategic thinking.
- No randomness — deterministic piece sequences

### Marathon (Worlds 76–80)
- 200+ line objectives. Endurance play.
- Progressive difficulty within single levels
- Multi-phase bosses with heal mechanics

## Engagement Features

### World Unlock System
- Worlds 1–10: Available from start (star-gated progression)
- Worlds 11–30: Seasonal worlds — unlock by having 100+ total stars
- Worlds 31–60: Campaign worlds — unlock progressively (need X stars)
- Worlds 61–80: Expert — unlock after completing all of 1–60
- Worlds 81–90: Special events + challenges
- World 100: All 99 worlds 3-starred

### Star Requirements (proposed)
| World Range | Stars to Unlock Next |
|-------------|---------------------|
| 1–10 | Original system (20/world) |
| 11–15 | 100 total |
| 16–20 | 200 total |
| 21–25 | 350 total |
| 26–30 | 500 total |
| 31–40 | 650 total |
| 41–50 | 900 total |
| 51–60 | 1200 total |
| 61–70 | 1500 total |
| 71–80 | 1800 total |
| 81–90 | 2000 total + special conditions |
| 100 | 5400 (all 3★) |

### Addiction Loop
1. **Short sessions**: Each level takes 1–4 minutes
2. **Star hunt**: 3-star every level for completionists
3. **Daily challenges**: Keeps players returning daily
4. **Streak rewards**: Consecutive days = bonus coins
5. **Seasonal FOMO**: Limited-time worlds with exclusive rewards
6. **Leaderboards**: Compete globally per level
7. **Achievement badges**: 200+ achievements across all worlds
8. **New piece mastery**: Each world pack teaches new pieces gradually
9. **Boss battles**: Epic encounters every 5th world (level 20)
10. **Share moments**: Share amazing clears/scores to social media
